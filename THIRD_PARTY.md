## Zenarchist's Fire Sticks

- Author: Zenarchist
- Workshop: https://steamcommunity.com/sharedfiles/filedetails/?id=3498006019
- Source: https://github.com/ZenarchistCode/ZenFireSticks

Zellno Safezone Campfire uses Zenarchist's Fire Sticks as an external dependency. It does not include, redistribute, or repack the original PBO, source code, signature, or key.

The dependency was audited during development. Its published description explicitly permitted repacking and modification when checked on 2026-07-17. This project deliberately uses the original mod as a dependency so upstream updates remain available.

All rights to Zenarchist's Fire Sticks remain with its original author.
