# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-18

### Added

- Configurable permanent administrative campfires.
- Multi-campfire JSON configuration and validation.
- Persistence-aware reconciliation without duplicate creation.
- Permanent fuel and automatic ignition.
- Optional weather immunity and disabled fire damage.
- Protection against extinguishing, cargo use, movement, and stone upgrades.
- Vanilla tripod, cooking pot, and cooking behavior.
- Zenarchist's Fire Sticks compatibility on unsupported terrain.
- FileBank and DSSignFile build pipeline.
- Local installation and client-link maintenance workflow.
